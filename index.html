    Welcome to the RingRoad Hospital CRC Zero-Feature Release (ZFR) milestone.

    Skip to end of metadata
    Go to start of metadata

    • RingRoad Hospital CRC is an open-source clinical data warehousing and analytics research platform. It enables sharing, integration, standardization, and analysis of heterogenous data from healthcare and research.
    • The RingRoad Hospital Community is a life-sciences-focused open-source, open-data community.  This website is the central place for the RingRoad Hospital Community to communicate and share projects with other users. Here you will find the latest information on the Software,installation guide,downloadable source code and find resources to help answer any questions you may have about the RingRoad Hospital CRC.
